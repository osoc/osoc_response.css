# osoc_response.css

自适配屏幕分辨率的层叠样式表

1. 加载本层叠样式表前，推荐先加载 [normalize.css](https://github.com/osoc/normalize.css/raw/3.0.3/normalize.css)；

2. 本样式表只定义根字体大小，其它节点的大小需要自行设置大小，推荐单位用‘em‘；

3. 代码语法高亮拓展：
